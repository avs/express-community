CB_ColorMap

Synopsis

Generate Colormaps from the Color Brewer database.

Parameters

Category UIOptionMenu type of colormap
Select Colormap by UIradioOption order to list available colourmaps
No of Colors / Name UIoptionMention number of colours or name of colormap
Stepped UItoggle colors a descrete if checked
colormap colormaps available

Output Ports

CB_Datamap Datamap datamap object of the selected Color Brewer colormap

Description

This macro generates highly intuitive colormaps that take into account human perception of colors, their tone and intensity. The macro uses the ColorBrewer database (a .csv file) and presents a list of colormaps from the users parameters. The user can then easily select the colormap closest to their need. ColorBrewer is an online tool for generating colormaps primarily looking at map data but can be used with any type of data and provides much better interpretation of the data than the standard visualization colormaps.

Parameters

Category
There are three categories qualitative(qual), sequential(seq) and diverging(div). Qualitative are for data that represents labels rather than quantitative values. Sequential for quantitative data that goes from a minimum to a max values. Diverging colormaps are for values that a based around some sort of average.
Select Colormaps by
Choose the colormap by its name or number of colors
No of Colors / Name
Select the number of colors or the name of the colormaps to list. This widget changes depending on what Select Colormaps by is set to.
Stepped
By default the colourmaps are continous ie the color for any value is interperted between the nearest key colors. Otherwise the colormap is descrete and values will be assigned the nearest key color.
Colormaps
A visual list of the colormaps available from the choices made previously. These are presented in a descrete form so the key colors can be clearly seen.

Output Ports

CB_Datamap
Standard Datamap object which contain the colormap create from the Color Brewer database. This object should be used as the Datamap input for the desired DataObject.
