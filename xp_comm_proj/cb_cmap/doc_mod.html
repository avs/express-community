CB_ColorMapCore

Synopsis

Read the ColorBrewer database from file and find the colormaps matching the required parameters. 
	group color_map {
		int ncolors;
		string category;
		string name;
		int colors[ncolors][3] ;	
	};

	module CB_ColorMapCore {
		string+IPort2 cbFile = "xp_comm_proj/cb_cmap/ColorBrewer_all_schemes_RGBonly3.csv";
		int ncats;
		string categories[ncats];
		int catidx;
		int mapidx;
		cmap_type_sel selector;
		omethod+notify_inst+req inst (
			cbFile+read+notify+req,
			ncats+write,
			categories+write+nosave,
			cmapptr+write+nosave
		)="CB_ColorMap_Inst";
		ptr cmapptr;
		omethod+notify_inst+req update (
			cmapptr+read+req,
			selector+read+notify,
			catidx+read+notify,
			mapidx+read+notify,
			ncatmaps+write,
			catmaps+write,
			ncmap+write,
			colormaps+write
		)="CB_ColorMapCore";
		omethod+notify_deinst+req destroy (
			cmapptr+read+req
		)="CB_ColorMap_Destroy";
		int ncatmaps;
		string catmaps[ncatmaps];
		int ncmap;
		color_map colormaps[ncmap];
	};

Description

Read the ColorBrewer database from file and find the colormaps matching the required parameters.
ColorBrewer is an online tool for generating colormaps primarily looking at map data but can be used with any type of data and provides much better interpretation of the data than the standard visualization colormaps.

Inputs

selector
Select on name or number of colors.
catidx
Category index (0 to ncats-1)
mapidx
Name index of colormap type
ncatmaps
Number of colours in selected colormaps

Outputs

ncmap
Number of colormaps matching requirements
colormaps
Array of the colomaps matching the requirements.

Example

CB_ColormapEg

Files

xp_comm_proj/cb_map/cbcmmods.v contains the color_map and CB_ColorMapParams groups and the CB_ColorMapCore module V definitions.
