FFTw

Synopsis

Perform the ``fastest Fourier transform in the west'' (FFTW) on the data in a uniform field.

Input Ports

in Mesh_Unif+Node_Data input field for fast Fourier transform
parent UIconnection user interface parent connection

Parameters

direction boolean selects forward/inverse fast Fourier transform
rescale integer selects type of rescaling of output field

Output Ports

out Mesh_Unif+Node_Data output field, fast Fourier transform of input field

Description

FFTw performs the ``fastest Fourier transform in the west'' (FFTW) on the data in a uniform field. FFTW is usually faster (and sometimes much faster) than all other public-domain Fourier Transform programs. The module works on Node_Data of any type or dimension. The dimensions do not have to be powers of two. The output field is a real (double), complex (vector length = 2) uniform field with the same dimensions as the input field.

Input Ports

in

The input field for the FFTw macro. A forward/inverse fast Fourier transform is performed on the Node_Data by the low-level module FFTw_raw.

Parameters

direction

A boolean that selects a forward/inverse (0/1) fast Fourier transform.

rescale

Depending on the value of rescale the output field is scaled as follows:

0 - No scaling
1 - Product of dimensions
2 - Sqare root of product of dimensions

If a forward and inverse FFT are connected then use the following pairs of rescale values (0, 1), (1, 0) or (2, 2) to ensure that the result is scaled correctly.

Output Ports

out

A real (double), complex (vector length = 2) uniform field with the same dimensions as the input field. The Node_Data is the fast Fourier transform of the input field.

Utility Macros

This User Macro combines the Functional Macro FFT with the UI Macro FFTw_UI, via the use of the common parameter group FFTw_params, to provide a usable macro in the visualisation library.

Example

An example application FFTwEg is provided that uses the AVS/Express sample data file hydrogen.fld and performs a forward and inverse FFT and displays the result.

Files

xp_comm_proj/fftw/FFTw_macs.v contains the Functional Macro FFT, the User Macro FFTw, the UI Macro FFTw_UI and the Example Application FFTwEg.

Prerequisites

This project relies on the successful installation of the following components. Without them it will not function correctly.
  • FFTW library
  • cp_utils project

Other Notes

The FFTwMacs library inherits its process. As this library contains no procedural code, the process is not important. The low-level FFTw_raw module, executes under the process specified in its library, not the process defined in the high-level library.
