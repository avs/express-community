GenerateStats

Synopsis

Generates the mean and standard deviation from data produced by the GeneralHistogram module.
module GenerateStats {
       prim binval<NEportLevels={2,1}>[];
       int binfreq<NEportLevels={2,1}>[];
       double mean<NEportLevels={1,2}>;
       double std_deviation<NEportLevels={1,2}>;
     omethod+notify_inst+req update<src_file="genstats.c">(
          binval+read+notify+req,
          binfreq+read+notify+req,
          mean+write,
          std_deviation+write ) = "generatestats";
};

Description

Generates the mean and standard deviation from a set of values and their frequencies as produced by the GeneralHistogramNodeData module.
If the data used to generate the histogram was iInteger data, then the mean and standard deviation are correct.
If the data used to generate the histogram was real data, then the mean will be correct (because of the way the GeneralHistogram module labels the frequencies) however the standard deviation is only approximate.

Inputs

binval[]
Array of type primative which holds the labels to the frequency values
binfreq[]
Integer array holding the frequencies generated.

Outputs

mean
Double precision floating point values, holding the mean of the input data.
std_deviation
Double precision floating point values, holding the standard deviation of the input data.

Utility Macros

The low level GenerateStats is used in the Functional Macro GeneralHistogram and the User Macro general_histogram. Additionally, these macros also use the GeneralHistogramParams parameter block group object, as does the GeneralHsitogramUI UI Macro and the GenerateStats Module.

Example

An example application GeneralHistogramEg is provided which uses the generalgraph module to display the histogram generated from the AVS/Express sample data file, lobstr.fld. The statistics are displayed in the user interface.

Files

xp_comm_proj/gnrlhist/his_mods.v contains the GeneralHistogramParams group, the GenerateStats module and the GeneralHistogramNodeData module V definitions.
The C code for the module is in file genstats.c
