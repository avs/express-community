NullBrickNodeData

Synopsis

NullBrickNodeData modifies a sub-volume of node data within a uniform volume, by replacing all the values with a single given value. This value may optionally be set to be the null_value. 
module NullBrickNodeData <src_file="nullbrik.cxx"> {
  Node_Data_Struct &in <NEportLevels={2,0}>;

  NullBrickParams_ref params <NEportLevels={2,0}>;

  int xmin => params.xmin;
  int xmax => params.xmax;
  int ymin => params.ymin;
  int ymax => params.ymax;
  int zmin => params.zmin;
  int zmax => params.zmax;
  double nval => params.nval;
  int setnull => params.setnull;

  Node_Data_Struct out <NEportLevels={0,2}> {
    &nnode_data => in.nnode_data;
    &ndim => in.ndim;
    &dims => in.dims;
  };

  cxxmethod+req update (
    in+read+notify+req,
    xmin+read+notify+req,
    xmax+read+notify+req,
    ymin+read+notify+req,
    ymax+read+notify+req,
    zmin+read+notify+req,
    zmax+read+notify+req,
    nval+read+notify+req,
    setnull+read+notify+req,
    out+write );
};

Description

NullBrickNodeDara inputs a uniform volume (Node_Data_Struct) and a set of parameters to define a sub-volume within, and a replacement data value. All node data within the sub-volume is replaced with the given value, in all components. This allows sub-volumes to be cut away - similar to the excavate_brick module - but doesn't convert the field to an unstructured mesh, thus allowing the volume to be volume rendered.

Additionally, the given value, may be specified as the null_value in the Node_Data components, and the null_flag parameter is set.

Inputs

&in
Input for uniform node data (Node_Data_Struct), which will have a sub-volume modified as defined by the extents in the parameter block, for all components of all primitive data types.
¶ms
Reference to the parameter block, which contains real instances of:

Parameters

xmin
Sub-volume lower extent for the x-axis.
xmax
Sub-volume upper extent for the x-axis.
ymin
Sub-volume lower extent for the y-axis.
ymax
Sub-volume upper extent for the y-axis.
zmin
Sub-volume lower extent for the z-axis.
zmax
Sub-volume upper extent for the z-axis.
nval
Value (null) to be used to fill all values defined as the sub-volume.
setnull
Sets the nval parameter to be the null_value in each component, and sets the null_flag of the Node_Data.

Outputs

out
Modified uniform Node Data output.

Utility Macros

NullBrickNodeData is used within the functional macro NullBrick and subsequently the high-level user macro null_brick.

Example

Two example applications, NullBrickEg1 and NullBrickEg2 are provided, which show the null_brick macro in use with the volume_render modules and the isosurface modules respectively.

Files

xp_comm_proj/nullbrik/nbrkmods.v contains the NullBrickParams group and the NullBrickNodeData module V definitions.

Other Notes

The low-level NullBrickMods library containing the module NullBrickNodeData does not specify a process. By default the express process will be used.
