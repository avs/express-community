Introduction

The STM codes are a suite of application codes and AVS modules available to display both geometric structure and electronic properties. Electronic and geometrical structural information is obtained from either Density Functional Theory (DFT) or Schrödinger Theory, using a variety of computational or experimental approaches. For more information see the design outline of STM plus the original AVS/Express paper presented at the AVS97 User meeting.

The STM codes are a suite of AVS/Express modules and ImmersadeskTM applications. STM can display geometric structure and electronic properties of molecules.

The AVS/Express modules require the AVS software from Advanced Visual Systems, Inc.. These modules have been tested on a Windows 95, Windows 98, Windows NT 4.0, SGI (both sg6 and sgN32 versions of AVS/Express) and the beta version of AVS/Express for Linux.

The Immersadesk application code requires the ImmersadeskTM display system from Pyramid Systems, Inc.
