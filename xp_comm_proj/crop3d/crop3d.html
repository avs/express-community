Crop3D

Synopsis

The Crop3D module provides a crop box module for unstructured fields with non polygonal cell sets.

Input Ports

Name Type Description
in Field Unstructured Field to be cropped.

Parameters

Name Type Description UI Control
xmin float minimum X-value for the crop box Slider
xmax float maximum X-value for the crop box Slider
ymin float minimum Y-value for the crop box Slider
ymax float maximum Y-value for the crop box Slider
zmin float minimum Z-value for the crop box Slider
zmax float maximum Z-value for the crop box Slider

Output Ports

Name Type Description
out Field Unstructured Field which is cropped to the given crop box
obj DataObject directly renderable object

Description

This project provides a crop box module for unstructured fields with non polygonal cell sets. The Crop3D module crops the Field (1D, 2D or 3D) within a region given by the minimum and maximum values of the crop box. The crop of a particular cell will happen only if all coordinates are defined after the crop.

Inputs

in
Unstructured Field with non polygonal cell sets that is to be cropped.

Parameters

minx
Float parameter that defines the minimum X-Value for the crop box.
maxx
Float parameter that defines the maximum X-Value for the crop box.
miny
Float parameter that defines the minimum Y-Value for the crop box.
maxy
Float parameter that defines the maximum Y-Value for the crop box.
minz
Float parameter that defines the minimum Z-Value for the crop box.
maxz
Float parameter that defines the maximum Z-Value for the crop box.

Outputs

out
Unstructured field that is cropped to the specified crop box.
obj
Directly renderable object of the cropped out field. This Port can be connected directly to the UViewer.

Utility Macros

The Crop3DFunc functional macro uses the Crop3DCore low-level module. The Crop3D user macro uses this functional macro and the Crop3DUI user interface macro. All of these macros use the Crop3DParams parameter block to communicate parameters between themselves.

Example

An example application is provided with the Crop3D project.

Files

xp_comm_proj/crop3d/crp3dmac.v Creates the Crop3DMacs library containing the user interface macro Crop3DUI, The functional macro Crop3DFunc and the high-level user macro Crop3D.

xp_comm_proj/crop3d/crp3dapps.v Creates the Crop3DApps library containing the example application Crop3DEg.
