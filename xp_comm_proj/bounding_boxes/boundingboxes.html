Boundingboxes

Synopsis

This module computes a boundingbox and provides a plane within this box to be used as cut-plane in other modules. It also allows to connect a secondary, deformed mesh to create a second bounding box.

Input Ports

Name Type Description
in_field Mesh_Unif+Node_Data input field for which the bounding boxes are computed
in_deformed Mesh_Unif+Node_Data optional secoundary, deformed Mesh

Output Ports

Name Type Description
out_boxes DefaultObject Directly renderable object of the 3D output
out_plane Mesh_Unif Mesh of the plane to be used as cut-plane in later other modules

Description

The Boundingboxes module generates a cubic boundingbox around the input-mesh. It provides a plane inside the bounding box which can be controled in the viewer. This plane is exported to be used as cut-plane or slice plane in other modules.

Parameters

Inputs

in_field
Field that contains the Mesh and Node_Data.
in_deformed
Optional, Field that contains the Mesh and Node_Data of the deformed object.

Outputs

out_plane
Mesh of the computed cut-plane
out_boxes
Directly renderable object that can be used with the UViewer3D.

Parameters

Show Deformed Bounds
This toggle activates the deformed boundinbox. It is available, when a Mesh is connected to in_deformed.
Enable Slice Plane
This toggle activates the cut-plane.
Axis
x, y, z or Free. Radiobox, that indicates the direction of the cut-plane. Free means the plane can be freely rotated inside the bounding box.
Plane
Slider, that controls the position of the cut-plane for x, y or z. Inactive for the freely transformed plane
Cut
Toggle, that indicates whether the plane is defining a cut-plane and therefore defines a new volume or if it is a plane on its own.
Cut above
Toggle, that indicates wether the boundingbox is cut above or below the plane.

Example

BoundingboxesEg1

This example application shows an object being cut and displayed with the two boundingboxes. A Read_Field is used to generate the input mesh. Cut and isovolume cuts this mesh with the plane generated by the boundingbox module.

BoundingboxesEg2

The secound example application contains additionally the first application a bounding box for the deformed data.
