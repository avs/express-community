DrawCrop

Synopsis

Allows the user to draw a box around the area of interest on each of the three orthoslices so the region can be cropped scaled and used in further networks.

Input Ports

field fieldin Uniform Scalar 3D Field
link DefaultView DefaultView

Output Ports

out_obj Principaloutput 2D renderable object
out_obj Secondaryoutput 2D renderable object
out_obj Tertiary output 2D renderable object
field_3DFld Uniform Scalar Field this is a 3D field
out_obj_3DDataObject output renderable object

Description

Allows the user to draw a box around the area of interest on each of the three orthoslices so the region can be cropped scaled and used in further networks. The module is high level and complex it was originally designed for a stand-alone application used to examine uniform medical scan data which has a different resolution along the z axis to the other two axes.

Input Ports

fieldin
This is connected to all the major components of the macro DrawCrop.
DefaultView
This is connected to the curr_view of the Uviewer2D, this is nescessary for the ContinuousSketch subcomponent.

Output Ports

Principal
The output field (2D uniform scalar field) as a renderable object.
Secondary
The output field (2D uniform scalar field) as a renderable object.
Tertiary
The output field (2D uniform scalar field) as a renderable object.
_3DFld
The output field is a 3D uniform scalar field, it may be scaled along the z-axis.
_3DDataObject
The output field (3D uniform scalar field, it may be scaled along the z-axis) as a renderable object.

Utility Macros

The User Macro DrawCrop combines the Functional Modules crop_controls, crop_data and crop_coordinate with a number of other macros. The macroUI defines the user interface while the macros box, ContinuousSketch, macroAXIS and macroAXIS2 are all small components of the functional macro macroDrawCrop which is combined with the UI, parameters and some other template macros to give the fully functional macro.

Example

An example application DrawCrop_Eg is provided, which reads in the lobster data set a uniform medical scan which can be scaled only in the z axis through this module.
The application produces three Views one for each axis of the orthoslice which are displayed in a 2D Viewer, the planes displayed can be altered through the UI. The box that shows the extents of the cropping are also displayed on these 2D Views.
The output from the module is a 3D cropped field which can be used as the input to another network, in this case one that produces an isosurface.

Files

xp_comm_proj/drawcrop/cropmacs.v contains several of the smaller components of the module box, ContinuousSketch, macroAXIS, macroAXIS2, parameters, macroUI, macroDrawCrop, DrawCrop and the example application DrawCrop_Eg.
Only the larger macros, DrawCrop and DrawCrop_Eg, are available through the IAC library the other smaller ones can be accessed through the templates library. This is because alteration of these components will only be of interest to developers who wish to build a stand alone application.
