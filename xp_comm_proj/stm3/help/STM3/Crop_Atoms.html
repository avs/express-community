Crop Atoms

Description

Crop the molecule inside a box, cylinder or sphere.

Input Ports

molecule MolecularDataType Molecular data.

Parameters

Crop type list Select the kind of cropping:
block
An axys oriented brick.
cylinder x
A cylinder oriented along the X axys.
cylinder y
A cylinder oriented along the Y axys.
cylinder z
A cylinder oriented along the Z axys.
sphere
A sphere.
reverse toggle Reverse inside and outside.
show geometry toggle Make visible the geometry on the geometry output.
add cross bonds toggle Add atoms just outside the geometry that have bonds with selected atoms.
Geometry various fields Set the geometry parameters specific for each shape.

Output Ports

inside MolecularDataType Selected molecular data inside the geometry.
outside MolecularDataType Selected molecular data outside the geometry.
geometry DefaultObject The cropping geometry.
