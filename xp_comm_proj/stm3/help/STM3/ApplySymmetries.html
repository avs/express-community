Apply Symmetries

Description

If the input molecule contains symmetry space group info the atoms are duplicated according to the space group symbol. If the resulting molecule contains duplicated atoms they are removed.

Input Ports

molecule_in MolecularDataType Molecular data with space group symmetry info.

Parameters

None

Output Ports

molecule_out MolecularDataType Molecular data with replicated atoms.
