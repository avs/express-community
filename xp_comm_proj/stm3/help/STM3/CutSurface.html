Cut Surface

Description

This is the standard AVS/Express cut_plane module. Respect to the original one has the following modifications:

  • Use the STM3 cache dimension settings.
  • Removed unused input port.
  • Fixed menu naming.

Cut a surface (eg. a Solvent Excluded Surface) with a plane.

Input Ports

in_field Field The field object that must be cut.

Parameters

The standard one for AVS/Express cut_plane module.

Output Ports

out_fld Mesh+Node_Data The resulting surface.
out_plane Mesh_Unif The cut plane.
out_obj_cut DefaultObject The renderable version of out_fld.
out_obj_plane DefaultObject The renderable version of out_plane.
