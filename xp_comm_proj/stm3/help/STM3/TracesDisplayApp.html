TracesDisplayApp

Description

This is a full application ready for use to display traces (accumulated positions) for specific atoms in a molecule traiectory. It combines the following modules:

  • Read Anim Molecule.
  • Display Molecule.
  • MolViewer.
  • Accumulate Traces.
  • The standard volume render AVS/Express module.
  • make movie.

This application already integrates the module needed to produce movies.
