make movie

Description

This is a modified image_capture AVS/Express module. Respect to the original one has the following modifications:

  • Fixed the naming in the module menu.
  • Fixed the capture mode to memory.
  • Removed unneeded ports.

This module capture a sequence of frames (one for each different viewer content) and assemble them in a movie (AVI or MPEG).

Input Ports

LGDView GDview_ref_templ The output view from MolViewer.

Parameters

The standard one for AVS/Express image_capture module.

Output Ports

None
