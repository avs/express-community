Miller Plane

Description

Visualize the plane identified by given Miller indices. Optionally remove atoms above or below it.

Input Ports

molecule_in MolecularDataType Molecular data with unit cell information.

Parameters

Miller plane indices field The Miller indices (hkl) identifying the plane. The plane (000) is invalid. The bar-n index is entered as negative number.
offset slider Plane displacement along the normal. The value is the fraction of the longest displacement along the normal that remains inside the unit cell.
scale u slider Scaling of one side of the resulting plane. This parameter modifies only the visualized plane.
scale v slider Scaling of the other side of the resulting plane. This parameter modifies only the visualized plane.
remove atoms above plane toggle Reverse the side of the plane from which the atoms are removed.
opacity slider Opacity of the resulting plane.

Output Ports

plane_fld Mesh The Miller plane.
plane_obj DefaultObject The renderable version of plane_fld.
out MolecularDataType Molecular part that falls above or below Miller plane.
