output image

Description

This is a modified OutputImage AVS/Express module. Respect to the original one has the following modifications:

  • Fixed the naming in the module menu.
  • Fixed a control panel sizing bug.
  • Set the image format to JPEG.
  • Set the image size to 1024x1024.

This module saves an image file with the content of the current viewer.

Input Ports

view_in GDview_templ The output view from MolViewer.

Parameters

The standard one for AVS/Express OutputImage module.

Output Ports

None
