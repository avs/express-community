NCAC Tube

Description

Compute the N-CA-C tube. The tube selects the atoms marked CA but pass only through the ones placed between a N and a C atoms.

Input Ports

molecule MolecularDataType Molecular data.

Parameters

Points added slider Points added between two CA intersection nodes.
Smoothness slider "Tension" parameter of the interpolating spline: a value of 0 creates stright tube segment between nodes; 1 is the normal value and more accentuate curvature.
Tube scale slider Radius of the tube.
Tube facets slider Number of sides composing the tube wall.

Output Ports

fld Mesh The tube itself.
obj DefaultObject The renderable version of fld.
