MolViewer

Description

This is a specially modified Uviewer3D AVS/Express module. Respect to the original one has the following modifications:

  • Viewer background color set.
  • The interactor works on the whole scene. This means that if you select a graphical object and then transform it, the whole scene is transformed.
  • The viewer size (512x512) is suited for movies generation.
  • Unused output ports are removed.

Input Ports

child_objs DefaultObject The set of graphic objects to be visualized.

Parameters

The standard one for AVS/Express Uviewer3D

Output Ports

picked_obj DefaultObject The picked graphic object. Picking is done by Control-LeftButton click on the viewer.
curr_view DefaultView The visualized scene.
