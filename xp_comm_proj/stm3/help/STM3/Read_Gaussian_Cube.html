Read Gaussian Cube

Description

Read the electronic density part from a Gaussian cube file. The resulting AVS/Express field can be visualized using standard techniques like isosurface or orthoslice.

Input Ports

None

Parameters

File name file browser The file to be read.
Units list Measurement units used in the files.

Output Ports

out Field_Unif The cube data. Beware that the uniform field has been skewed, using xform, to match the Gaussian Cube cell size.
