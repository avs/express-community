Color Legend

Description

Display a legend for the molecule representation colors. There are three types of display:

  • The list of unique atoms with an example of atom sphere.
  • The list of residues.
  • A continuous representation of the color scale for atoms charge values.

Input Ports

display_params DisplayParams Molecule display settings data.

Parameters (for atom and residue colors)

X min, X max various numeric fields The horizontal extension of the legend. The screen goes from -1 to 1. Currently X max is ignored.
Y min, Y max various numeric fields The vertical extension of the legend. The screen goes from -1 to 1.
Z value numeric field The Z position of the legend. The screen goes from -1 to 1.
Scale numeric field The scale of the spheres representing atoms and residues.
Label offset numeric field The offset of the label respect to the corresponding sphere.
Font font selector Standard AVS/Express text parameters selector.

Parameters (for atom charge)

The parameters are the same of the standard AVS/Express LegendVert module.

Output Ports

legend_obj DefaultObject Graphic object to be sent to the viewer (MolViewer or the standard Uviewer3D).
