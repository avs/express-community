Display Unit Cell

Description

Displays the limits of the unit cell. Nothing is displayed if the molecular data does not contains unit cell data.

Input Ports

in MolecularDataType Molecular data containing unit cell definition data (ie. read from a format that contains it).

Parameters

None

Output Ports

fld Mesh+Node_Data The unit cell as an Hex volume cell. The node data is set to RGB white.
obj DefaultObject The renderable version of fld. The render properties set only the Hex cell borders as visible.
