Find Tetrahedra

Description

Display tetrahedra around specific atom types. The atom must have 4 connected atoms that will be used as vertices of the tetrahedra.

Input Ports

molecule_in MolecularDataType Molecule.

Parameters

Select tetrahedra center atoms multi select list The list of atoms contained in the molecule that have exactly 4 bonds.
Deselect All button Deselect all the entries in the Select tetrahedra center atoms list.
Select All button Select all the entries in the Select tetrahedra center atoms list.
Selection type radio buttons Select the kind of operation:
exact selection
Draw tetrahedra for only the atoms selected in the list
select similar similar
Draw tetrahedra for the atoms with names that starts like to the ones selected in the list

Output Ports

obj DefaultObject The renderable version of tetrahedra.
