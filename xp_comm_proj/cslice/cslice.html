CoordinatesSlice

Synopsis

This module extract points from a given array with 3D-points, which are positioned on a defined plane. The extracted points are output as 2D-points.

Inputs

Name Type Description
Coords_3D Coords_3D group containing array with 3D-points and dimension of the array

Parameters

Name Type Description UI Control
plane float value of plane-coordinate Slider
axis int x-,y-, or z-axis (equivalent to values 0, 1, 2) Slider
dec int number of decimals of plane-value

Output Ports

Name Type Description
xy_out array [][2] array of 2D-points matching the plane

Description

The module extracts points from an array of 3D-points, which are positioned on a specified plane. The plane is defined by two values:
  • Axis, which specifies the alignment of the plane through the values 0,1 or 2, equivalent to x-, y- or z-axis (the axis orthogonal to the plane).
  • Plane, which defines the distance between the origin of the coordinate system and the plane
Both of these values can be selected by sliders.
The slider of the Plane-value is limited by the maximum and minimum values of coordinates at the specified axis.
The precision of the result depends on the precision of the Plane-value. The 3D-points are truncated to the same number of decimals as the number of decimals of the Plane-value.
The module outputs an array of 2D-points consisting of the matching points neglecting the values of the coordinate specified by Axis.

Parameters

Inputs

Coords_3D
group containing the following parameters:
  • dimension, which specifies the number of the 3D-points
  • xyz_Coords, an array [dimension][3] containing the coordinates of the 3D-points
axis
Alignment of the plane, values 0,1 or 2 for x-,y- or z-axis
plane
Value specifying the exact position of the plane

Outputs

xy_out
array [][2] of the points matching the plane

Example

The example contains a sphere, Coords_3D containing the array of the points of the sphere and the dimension of this array, a point_mesh to convert the result of the module and a Uviewer3D to visualize the result.
The CoordinatesSlice-Module provides the User-Interface for selecting the Axis- and Plane-values and outputs the points matching the plane.
