read_BioRad_PIC

Synopsis

Reads a Bio-Rad PIC file and outputs a uniform scalar field.

Parameters

Name Type Description UI Control
filename string name of the PIC file to be loaded UIfiledialog

Output Port

Name Type Description
out_fld Mesh_Unif+Node_Data output field containing the data in the PIC file.
out_obj DataObject directly renderable output object.

Description

This module reads a Bio-Rad PIC file specified by the user via a file browser, and outputs a field corresponding to the data in the file. Bio-Rad PIC files contain 3D confocal microscope images, i.e. 3D scalar uniform fields. read_BioRad_PIC outputs a 3D scalar uniform field and a renderable version of the field.
This module performs the byte reordering necessary for it to work correctly on either big or little endian platforms.

Parameters

filename
The filename of the Bio-Rad PIC file that should be read.

Output Ports

out_fld
The output 3D scalar uniform field generated from the data in the Bio-rad PIC file.
out_obj
The directly renderable output of the read_BioRad_PIC macro. This contains a DataObject object that can be rendered by the standard data viewers.

Utility Macros

The User Macro read_BioRad_PIC combines the Functional Macro Read_BioRad_PIC and the User Interface Macro Read_PIC_UI. In turn the Functional Macro Read_BioRad_PIC combines the low-level module read_PIC_core with the field mapping module, scalar_uniform_field.

Example

An example application Read_BioRad_PIC_Eg is provided that reads a Bio-Rad PIC file and produces a maximum intensity projection. A sample Bio-Rad PIC file, flea3.pic, is provided with the distribution of this module.

Files

xp_comm_proj/PIC_IO/pio_macs.v contains the User Macro read_BioRad_PIC, the Functional Macro Read_BioRad_PIC, the User Interface Macro Read_PIC_UI and the Example Application Read_BioRad_PIC_Eg.

Other Notes

The read_BioRad_PIC macro does not check that the file specified by the user is a valid Bio-Rad PIC file. If it is not then errors can occur.
The read_BioRad_PIC macro ignores notes, which are sometimes appended to PIC files.
The read_BioRad_PIC macro will only handle 8-bit images. 16-bit and 24-bit images are not handled correctly.
The AVS5 module read_BioRad_PIC (available from the IAC) can also be used to read Bio-Rad PIC files.
This project does not contain any C or Fortran code and hence does not need to be compiled.
