downsample_mean

Synopsis

The downsample_mean macro reduces the size of uniformed mesh data.

Input Ports

Name Type Description
in Mesh_Unif + Node_Data Required data field

Parameters

Name Type Description UI Control
output_size[3] int Size of the desired mesh UIsliders

Output Ports

Name Type Description
out_fld Mesh_Unif + Node_Data Reduced data field
out_obj GDobject Directly renderable output object

Description

The downsample_mean macro reduces the size of the input data field to the size provided by the user using the Sliders in the user interface. The data reduction is done by calculating the mean value of all original points around the new data point. It can only handle uniform mesh data.

Inputs

in
The original Uniform Mesh.

Parameters

output_size[3]
Defines the desired output size of the Mesh. New values of the field are calculated as the mean of all surrounding points which fall away through these data reduction.

Utility Macros

The functional macro DownSampleFunc uses the low-level module downsample. The user macro downsample_mean uses this functional macro and the user interface macro BackgroundUI. The BackgroundParams parameter block is used to connect these components together.

Example

The BackgroundEg1 example application reads a file and displays the reduced object.

The BackgroundEg2 example application reads a file, reduces its size, and and creates a isosurface with the reduced mesh.
